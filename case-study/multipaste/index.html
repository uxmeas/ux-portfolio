Deep Problem Framing
The Goal

Designers paste text into 50+ layers one by one — selecting a layer, pasting, selecting the next, pasting again. For a single table mockup, this takes 15 minutes of pure repetition with zero creative value.

The Problem

Figma has no native way to distribute structured data across multiple text layers. Every table, list, or data-heavy component requires manual copy-paste per cell. Designers waste hours on work that should take seconds.

How Might We

How might we let designers paste structured data into dozens of text layers in one action — eliminating 15 minutes of repetitive work per table?

Before 15 min manual paste Select layer, paste, next layer, paste. Repeat 50+ times per table mockup.
After 1 min with MultiPaste Paste data, select layers, click apply. One action for all layers.
User Research
Key Insights
Preview prevents mistakes

Showing which text goes to which layer before applying caught 100% of ordering mistakes. Without preview, designers would apply, see errors, undo, reorder, and try again.

Selection order matters

Designers select layers in visual order (top to bottom, left to right), but Figma's selection order doesn't always match. Making selection order predictable and visible was critical.

Empty values need handling

Real data has gaps — empty cells, missing fields, placeholder values. The plugin needed options to skip empty values or preserve existing text, not just blindly overwrite everything.

Design Approach / Solution

MultiPaste takes structured text — from spreadsheets, CSVs, or plain lists — and distributes each value to a selected text layer in Figma. One paste, one click, all layers populated.

The plugin prioritizes trust: a live preview shows exactly which text goes to which layer before anything changes. One-click undo reverts everything if the result isn't right.

Key Features

1. Multi-delimiter support (tab, comma, newline, pipe, custom)

2. Live preview — see the mapping before applying

3. One-click undo — revert all changes instantly

4. Trim whitespace and skip empty value toggles

Reflection
✓ What Worked

1. Scratching my own itch — I built this because I needed it, which meant instant validation and clear requirements from day one

2. Preview-first design — showing the mapping before applying eliminated the undo-retry loop that makes batch operations stressful

◈ What I'd Do Differently

1. Add spreadsheet import — direct .xlsx/.csv file upload would save the copy-paste step entirely

2. Analytics tracking — no telemetry means no visibility into which features are most used or where users get stuck
