CHALLENGE

THE
CHALLENGE

The workflow is universal and universally painful. A client sends a Word document. The designer opens it and begins manual labor: copy heading, paste into Figma, style it. Copy body text, paste, style it. Recreate the table structure from scratch. Repeat for 30 pages.

No existing Figma plugin solved the full document-to-design pipeline.

INSIGHT

THREE DISCOVERIES

DISCOVERY 01
One-Click Beats
Configurability
Plugins with extensive configuration had low adoption. Designers want to drop a file and get a result.
DISCOVERY 02
Plugin UI Is Its
Own Discipline
480×600px is not a web page, not a mobile screen. It demands its own design language.
DISCOVERY 03
Dual-Layer Parsing
Absorbs Chaos
A single parsing strategy cannot handle real-world .docx variance. Two layers compensate for each other.
WORKFLOW / 5-STEP WIZARD

THE WIZARD

Five steps from Word document to styled Figma design. Each step handles one clear decision. No configuration screens, no settings panels. Just forward momentum.

01
Upload
Drop a .docx file. The parser reads headings, body text, lists, and tables in one pass.
02
Preview
See your document structure before any conversion. Headings, paragraphs, and lists mapped to Figma layers.
03
Style
Pick a text style preset. Seven auto-generated styles map to your document hierarchy.
04
Generate
One click. Text layers, auto-layout frames, and styled components appear on your Figma canvas.
05
Done
Your document is now a Figma design. Edit, restyle, or hand off. The original structure is preserved.
DESIGN SYSTEM / PLUGIN VIEWPORT

COMPACT PRO

Designing for 480 x 600px

Figma plugin panels are 300-480px wide and 600px tall. Every pixel matters. I built a complete design system for this constraint: compact inputs, stacked controls, icon-only toggles, and a type scale that stays readable at 11px.

The constraint became the product's personality. Dense, precise, zero wasted space.

BRAND / IDENTITY

BRAND IDENTITY

The copper-to-teal gradient is the brand story. Copper represents the raw input. A Word document. Unformatted, unstructured. Teal represents the output. A styled Figma design. Clean, structured, ready to ship. The gradient connects them. Transformation in a single visual.
